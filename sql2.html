Querying Multiple Tables Using Joins

  • 1. JOINS
    • a) Equi Join (Inner Join)
    • b) Cross Join
    • c) Outer Joins
      • 1. Left Outer Join
      • 2. Right Outer Join
      • 3. Full Outer Join
    • d) Self Join

Query

Equi Join Clause



Syntax:

SELECT column1, column2, ...
FROM table1
INNER JOIN table2
ON table1.column = table2.column;


Ex:

mysql> SELECT Students.Name, Courses.CourseName 
FROM Students
INNER JOIN Courses 
ON Students.CourseID = Courses.CourseID;

Output

Table Before Join:


+--------+----------+---------+
| Name   | CourseID | CourseName |
+--------+----------+---------+
| Lucky  |   C1     | Math      |
| Ajay   |   C2     | Science   |
| Mahi   |   C3     | History   |
| Neha   |   C4     | English   |

4 rows in set (0.00 sec)

+----------+------------+
| CourseID | CourseName |
+----------+------------+
|   C1     | Math       |
|   C2     | Science    |
|   C3     | History    |
|   C4     | English    |
+----------+------------+
4 rows in set (0.00 sec)

After Applying Join:


+--------+-----------+
| Name   | CourseName|
+--------+-----------+
| Lucky  | Math      |
| Ajay   | Science   |
| Mahi   | History   |
| Neha   | English   |

Query

SELF JOIN Clause



Syntax:

SELECT A.column1, B.column2
FROM table_name A, table_name B
WHERE A.column = B.column;


Ex:

mysql> SELECT A.Name AS StudentName, B.Name AS MentorName
FROM Students A, Students B
WHERE A.MentorID = B.StudentID;

Output

Table Before Join:


+-----------+-----------+-----------+
| StudentID | Name      | MentorID  |
+-----------+-----------+-----------+
| 1         | Lucky     | 2         |
| 2         | Ajay      | 3         |
| 3         | Mahi      | NULL      |
| 4         | Neha      | 2         |
+-----------+-----------+-----------+
4 rows in set (0.00 sec)

After Applying SELF JOIN:


+--------------+-------------+
| StudentName  | MentorName  |
+--------------+-------------+
| Lucky        | Ajay        |
| Ajay         | Mahi        |
| Neha         | Ajay        |
+--------------+-------------+
3 rows in set (0.00 sec)
