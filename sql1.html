    • OPERATORS

    • 1. Comparison Operators
      • (=): Equal to
      • (!= or <>): Not equal to
      • (>): Greater than
      • (<): Less than
      • (>=): Greater than or equal to
      • (<=): Less than or equal to
      • (IN): Used to check if a value is in a list of values
      • (NOT IN): Used to check if a value is not in a list of values
      • (LIKE): Used to search for a specified pattern
      • (IS NULL): Used to check if a value is NULL
      • (IS NOT NULL): Used to check if a value is not NULL
    • 2. Logical Operators
      • (AND): Returns true if both conditions are true
      • (OR): Returns true if at least one condition is true
      • (NOT): Reverses the result of a condition
    • 3. Aggregate Operators
      • (COUNT()): Returns the number of rows
      • (SUM()): Returns the sum of a numeric column
      • (AVG()): Returns the average value of a numeric column
      • (MIN()): Returns the smallest value
      • (MAX()): Returns the largest value
    • 4. Set Operators
      • (UNION): Combines the results of two queries (removes duplicates)
      • (UNION ALL): Combines the results of two queries (keeps duplicates)
      • (INTERSECT): Returns only the rows that are common between the two SELECT queries
      • (EXCEPT/MINUS): Returns rows from the first query that are not in the second query

Query

Creating a foreign key with ON DELETE CASCADE



Syntax:

create table student (
    id int auto_increment primary key,
    name varchar(50)
);

create table enrollment (
    enrollment_id int auto_increment primary key,
    student_id int,
    course_name varchar(50),
    foreign key (student_id) references student(id) on delete cascade
);


Ex:

-- Create `student` table
create table student (
    id int auto_increment primary key,
    name varchar(50)
);

-- Create `enrollment` table with foreign key on `student_id`
create table enrollment (
    enrollment_id int auto_increment primary key,
    student_id int,
    course_name varchar(50),
    foreign key (student_id) references student(id) on delete cascade
);

-- Insert some sample data into the `student` table
insert into student (name) values ('John Doe');
insert into student (name) values ('Jane Smith');

-- Insert some sample data into the `enrollment` table
insert into enrollment (student_id, course_name) values (1, 'Math');
insert into enrollment (student_id, course_name) values (1, 'Science');
insert into enrollment (student_id, course_name) values (2, 'History');

-- Display data in the tables
select * from student;
select * from enrollment;

-- Delete a student (to test ON DELETE CASCADE)
delete from student where id = 1;

-- Display data again after the delete
select * from student;
select * from enrollment;

Output


-- Before delete
mysql> select * from student;
+----+--------------+
| id | name         |
+----+--------------+
|  1 | John Doe     |
|  2 | Jane Smith   |
+----+--------------+
2 rows in set (0.00 sec)

mysql> select * from enrollment;
+----------------+------------+-------------+
| enrollment_id  | student_id | course_name |
+----------------+------------+-------------+
| 1              | 1          | Math        |
| 2              | 1          | Science     |
| 3              | 2          | History     |
+----------------+------------+-------------+
3 rows in set (0.00 sec)

-- After delete
mysql> delete from student where id = 1;
Query OK, 1 row affected (0.01 sec)

mysql> select * from student;
+----+--------------+
| id | name         |
+----+--------------+
|  2 | Jane Smith   |
+----+--------------+
1 row in set (0.00 sec)

mysql> select * from enrollment;
+----------------+------------+-------------+
| enrollment_id  | student_id | course_name |
+----------------+------------+-------------+
| 3              | 2          | History     |
+----------------+------------+-------------+
1 row in set (0.00 sec)

Single-row Subquery



Syntax:

SELECT column_name

Ex:

mysql> SELECT name FROM employees
       WHERE salary = (SELECT MAX(salary) FROM employees);

| name       |
+------------+
| John Smith |
+------------+

Query

Multi-row Subquery



Syntax:

SELECT column_name
FROM table_name
WHERE column_name IN (SELECT column_name FROM another_table WHERE condition);


Ex:

mysql> SELECT name FROM employees
       WHERE department_id IN (SELECT department_id FROM departments WHERE location = 'New York');

Output


+------------+
| name       |
+------------+
| Jane Doe   |
| Alice Lee  |
+------------+
